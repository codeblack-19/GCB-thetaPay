
# Buysome E-commerce

An e-commerce application which was built with `NodeJs`, `express`, and `ReactJS`.
The system was built to test a payment gateway's web payment functionality.

## How to run the app

Before you can execute this program, first make sure you have nodejs running on your computer.

* After navigating into the project run the command `npm install` to install all the npm packages used to build the backend.

* Once all the npm packages are installed successfully, type `npm run build` in the terminal to build the React application in the client directory.

* When the build is complete, type `npm start` to start the application on.

* The application will run on port `3006`. Open `localhost:3006` in the browser to test the app.
